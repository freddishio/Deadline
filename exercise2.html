Mealy FSM for Patient Monitor Alarms

Combined 4-state model for heart-rate and blood-pressure alarms. Inputs are abbreviated in the legend below.

Input legend

  • h = heart rate above its limit
  • b = blood pressure above its limit

Output legend

  • HR+ = switch heart-rate alarm on
  • HR- = switch heart-rate alarm off
  • BP+ = switch blood-pressure alarm on
  • BP- = switch blood-pressure alarm off
  • - = no output action
The four states encode which alarms are currently active. This is a Mealy machine, so each transition label is written as input condition / output action.
